Generates a Hilbert matrix element.

This function generates an element for a Hilbert matrix. The element at position \((i, j)\) is:

\[ a_{i,j} = \frac{1}{i + j - 1} \]

where \(i\) and \(j\) are 1-indexed row and column indices.

Parameters
rowThe row index of the matrix element ( \(i\)).
colThe column index of the matrix element ( \(j\)).
Returns
The Hilbert matrix element.

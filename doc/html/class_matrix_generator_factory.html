Detailed Description

template<typename T>
class MatrixGeneratorFactory< T >

Factory class to generate different types of matrix generators.

This factory class is responsible for creating instances of matrix generators based on user input. It determines the appropriate generator type (e.g., from a function or file) and constructs it accordingly.

Template Parameters
TThe data type of the matrix elements (e.g. float, double).

Detailed Description

template<typename T>
class OutputGenerator< T >

Class for generating outputs of eigenvalue computations.

This class handles the output of eigenvalue results either by printing them to the terminal or saving them to a file, based on user-specified options.

Template Parameters
TThe data type of the eigenvalues (e.g. float, double).

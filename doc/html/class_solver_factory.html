Detailed Description

template<typename T>
class SolverFactory< T >

Factory class to generate different types of solvers.

This factory class is responsible for creating instances of solvers based on user input. It determines the appropriate solver and constructs it accordingly.

Template Parameters
TThe data type of the matrix elements (e.g. float, double).

std::unique_ptr< AbstractIterativeSolver< T > > SolverFactory< T >::ChooseSolver ( )

Select the right solver for the eigenvalue problem based on the method name specified by the user.

This method evaluates the method name (methodName) and create the solver with the right arguments (methodArgs). The method also checks that the method arguments are valid, and converts them to the right type.

Returns
A unique pointer to a solver object.

Detailed Description

template<typename T>
class InversePowerMethodSolver< T >

Class for finding an eigenvalue using the Inverse Power method.

This class extends the AbstractIterativeSolver base class by providing the inverse power solver method (with a shift to be chosen) to find the eigenvalue of a matrix.

When no shift is specified, this methods finds the smallest eigenvalue of the matrix. Otherwise, it finds the eigenvalue closest to the shift value.

Template Parameters
TThe data type of the matrix elements (e.g. float, double).

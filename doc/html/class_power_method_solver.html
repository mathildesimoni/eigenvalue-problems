Detailed Description

template<typename T>
class PowerMethodSolver< T >

Class for finding an eigenvalue using the Power method.

This class extends the AbstractIterativeSolver base class by providing the power solver method (with a shift to be chosen) to find the eigenvalue of a matrix.

The eigenvalue found by this method is the dominant eigenvalue. By choosing a value for the shift, the convergence of the method may be increased.

Template Parameters
TThe data type of the matrix elements (e.g. float, double).
